# 🌟 Fynix Centralized Wallet 🌟

[🌐 Open Frontend Web Page](https://tushar10-web.github.io/Fynix-Centralized-Wallet/))

## 🚀 About
Fynix Centralized Wallet is a **secure**, **easy-to-use** platform for managing multiple cryptocurrencies. It offers fast transactions, custodial key management, integrated trading, and strong security features like multi-factor authentication, making it ideal for both beginners and pros.

## 💡 Features
- 🔒 Strong security with multi-factor authentication  
- ⚡ Fast cryptocurrency transactions  
- 🔐 Custodial key management  
- 📈 Integrated trading platform  
- 👨‍💻 User-friendly interface for beginners and pros  

## 📁 Project Structure
- `frontend/` - Frontend web files including index.html  
- `backend/` - Backend server and API code  
- `README.md` - This README file  

## 🛠️ Installation & Usage
1. Clone the repository  
2. Navigate to the frontend folder to open `index.html` in your browser  
3. Run/setup backend server as per backend instructions
